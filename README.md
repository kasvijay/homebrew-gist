# homebrew-gist

Homebrew tap for [Gist](https://github.com/kasvijay/Gist) — a native macOS transcription app.

## Install

```bash
brew tap kasvijay/gist
brew install --cask gist
```

## Update

```bash
brew upgrade --cask gist
```

## Uninstall

```bash
brew uninstall --cask gist
```

Your recordings in `~/Documents/Gist` are left in place. To also remove app
preferences and caches, use `brew uninstall --zap --cask gist`.
